# homebrew-tap

Homebrew tap for [Ctrlplane CLI](https://github.com/ctrlplanedev/cli).

## Installation

```bash
brew tap ctrlplanedev/tap
brew install ctrlc
```

## Upgrade

```bash
brew upgrade ctrlc
```

## Uninstall

```bash
brew uninstall ctrlc
brew untap ctrlplanedev/tap
```

## About

This tap is automatically updated when new releases are published to [ctrlplanedev/cli](https://github.com/ctrlplanedev/cli).
